# Verify Start

This is an alias for `/xspec:verify:generate-witness`. Running `/xspec:verify:start` begins the verification workflow by generating witness scenarios — the natural first step after typechecking a spec.

Executing this command runs the same procedure as `/xspec:verify:generate-witness`. See that command for full documentation.
